# Copyright.sh AI License - WordPress Plugin

Official WordPress plugin for AI content licensing. Block AI crawlers or license your content and get paid.

[![WordPress Plugin Version](https://img.shields.io/wordpress/plugin/v/copyright-sh-ai-license)](https://wordpress.org/plugins/copyright-sh-ai-license/)
[![WordPress Plugin Downloads](https://img.shields.io/wordpress/plugin/dt/copyright-sh-ai-license)](https://wordpress.org/plugins/copyright-sh-ai-license/)

## Features

- Block AI crawlers via robots.txt (GPTBot, Claude, Gemini, etc.)
- Add machine-readable licensing meta tags for compliant AI companies
- Set pricing for AI training and inference usage
- Per-page/post license overrides
- One-click dashboard connection

## Installation

1. Download from [WordPress.org](https://wordpress.org/plugins/copyright-sh-ai-license/)
2. Or search "Copyright.sh AI License" in WordPress plugins
3. Activate and configure under Settings > AI License

## License Grammar v1.5

```html
<meta name="ai-license" content="allow; distribution:public; price:0.15; payto:your-account">
```
