Generate AI-usage <meta> tags and ai-license.txt to monetise content.

== Description ==

Easily declare, customise and serve machine-readable AI-usage licences for your WordPress site.

Generates `<meta name="ai-license">` tags and a domain-wide `ai-license.txt` endpoint. Includes a settings screen and per-page override panel.

== Installation ==
1. Upload the plugin ZIP via Plugins → Add New.
2. Activate.
